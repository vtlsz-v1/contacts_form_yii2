нужно создать форму создания и редактирования пользователя.

1. Создать форму с полями
- Имя
- Фамилия
- Номер телефона

2. Проверить поля формы
- Имя и фамилия не должны содержать пробелов или других символов кроме букв
- Телефон не должен содержать букв и других символов кроме цифр и не должен быть длиннее 10 символов
- Имя, фамилия и телефон не должны быть пустыми

3.  Сохранить данные в таблице базы данных.
Таблица должна содержать поле для:
- ID пользователя
- Имени
- Фамили
- Номера телефона
- даты создания
- даты обновления

4. После сохранения, переходить на страницу со списком всех пользователей, где будут выведены данные в колонках:
- ID пользователя
- имя
- фамилия
- номер телефона
- дата создания
- дата обновления
- ссылка на редактирование пользователя
- ссылка на удаление пользователя

5. ссылка на удаление, должна опросить "хотите удалить пользователя ID '#'?", после подтверждения удалить пользователя из базы и перезагрузить страницу на лист со всеми пользователями.

6. ссылка на редактирование должна вывести страницу с заполненной данными формой с полями из пункта "1" (имя, фамилия, номер). После повторного подтверждения формы обновлять данные в базе данных.

Это бекенд задача, оформление верстки можно сделать любым или воспользоваться готовыми наборами. Для обработки запросов нужно использовать yii2, так как на нём есть действующий проект.
